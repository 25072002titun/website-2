Syncano Hosting

The best way to manage your hosting files is through the syncano-cli. You should have Python installed. You should have Python installed. Learn how to install Python on Windows or Mac OS X.

Install syncano-cli:

$ pip install syncano-cli

Login to your Syncano account:

$ syncano login --instance-name YOUR_INSTANCE

Deploy your application

Deploying your app is as simple as:

$ syncano hosting publish ./your/project/path/

To see all the available commands:

$ syncano hosting

With Syncano-CLI, you can do so much more than just managing your static files. See the syncano-cli GitHub respository for a detailed usage instruction.
