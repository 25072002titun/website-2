Syncano Hosting

The best way to manage your hosting files is through the syncano-cli. You can install it easily through yarn or npm

Install syncano-cli:

$ yarn add syncano-cli

Initialize a new project

$ syncano-cli init

Deploy your application

Create a directory for your hostings:

$ mkdir 'directory_name'

Add a hosting:

$ syncano-cli hosting add [hosting_path]

Deploying your app is as simple as:

$ syncano-cli hosting sync 'hosting_name'

To see all the available commands:

$ syncano-cli hosting

With Syncano-CLI, you can do so much more than just managing your static files. See the Syncano-cli NPM respository for a detailed usage instruction.
